## How to start this ptoject!

# Note This project was created using node version **v17.4.0**

## clone the branch

```
git clone https://github.com/Milan-960/Holiday-search.git
```

## Install dependecies

```
npm install
```

## run on the localhost

```
npm start
```

## Assignment Criteria

‚óè Users should be able to search by free text

‚óè Search results should show the total number of results

‚óè Search results should only include the first 10 sales in the list

‚óè Each search result should link to the sale detail page

‚óè Browser navigation should work (e.g. going from search results to sale details and
clicking the back button should take you back to search results)

## All of the functionality has been added!

| Details        | Route                       | Description                                                                                                                                                                 |
| -------------- | --------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Homepage       | /                           | Simply contains a search box for a user's location query.                                                                                                                   |
| Search results | /search?query=<search term> | A list of results that match the search performed, and link through to the details page. The list should show: <br>‚úÖ Title<br>‚úÖ Destination<br>‚úÖ Lead photo              |
| Sale details   | /sale/<ID>                  | A page with more rich information about the hotel/sale in question. The page should include:<br>‚úÖ Title<br>‚úÖ Destination<br>‚úÖ Lead photo<br>‚úÖ Hotel details<br>‚úÖ Price |
